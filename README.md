# Solitario Card Game

Klondike Solitaire implementation in Next.js.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000/solitario-card-game](http://localhost:3000/solitario-card-game)

## How to Play

- Click the stock pile to draw cards
- Drag cards to build tableau columns in alternating colors, descending order
- Move Aces to foundations, then build up by suit to King
- Win by completing all 4 foundations (A through K per suit)
